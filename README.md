# Article Harness Portfolio Page

This workspace now contains a compact portfolio detail page for the `Article Harness` project.

## Page direction

- Chinese-first project detail page
- Compact, resume-like information density
- Focused on project summary, workflow, structure, and design decisions

## Main files

- `article/App.tsx`: page layout
- `article/data.ts`: editable project copy
- `article/styles.css`: visual style
- `docs/`: longer design and portfolio copy drafts

## Commands

```bash
npm install
npm run dev
npm run build
npm run html
```

`npm run html` exports a shareable single-file page to `article/article.html`.
